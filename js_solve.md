## MODEL ✨

在 JavaScript (.js) 文件中，有两种主要的注释方式：

### 1. 单行注释 (Line Comment)

使用两个斜杠 `//` 来创建单行注释。从 `//` 开始直到该行的末尾，所有的内容都会被视为注释。

**语法：**
```javascript
// 这是注释内容
```

**示例：**
```javascript
// 这是一条单行注释，用来解释下面的变量
let name = "Alice"; // 也可以在代码行的末尾添加注释

// 下面的代码被注释掉了，所以不会执行
// console.log("这行代码不会运行");
```

### 2. 多行注释 / 块级注释 (Block Comment)

使用 `/*` 开始，并以 `*/` 结束来创建多行或块级注释。介于 `/*` 和 `*/` 之间的所有内容（可以跨越多行）都会被视为注释。

**语法：**
```javascript
/*
  这是注释内容
  可以写很多行
*/
```

**示例：**
```javascript
/*
  这是一个多行注释（块级注释）。
  通常用于解释复杂的函数、文件用途，
  或者临时禁用一大段代码。
*/
function greet() {
  console.log("Hello, world!");
}

/*
下面的函数暂时不用，先注释掉
function oldFunction() {
  console.log("This is an old function.");
}
*/
```
